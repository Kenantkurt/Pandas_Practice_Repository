# Pandas Practice Repository

## Overview

This repository contains structured practice notebooks covering core pandas operations.  
The goal is to build strong fundamentals in data manipulation, transformation, analysis, and input/output workflows.

The notebooks follow a progressive learning path — starting from basic data loading and moving toward aggregation, time series analysis, and multi-source data handling.

---

## Topics Covered

### 01 — Data Loading
- Reading CSV files
- Inspecting DataFrames
- Basic exploration

### 02 — Row & Column Selection
- Selecting columns
- loc vs iloc
- Boolean indexing
- Conditional filtering

### 03 — Index Operations
- Setting and resetting index
- Multi-index basics
- Index-based selection

### 04 — Filtering & Conditional Selection
- Boolean masks
- Multiple conditions (&, |)
- isin() and between()
- Conditional column selection

### 05 — Updating Rows & Columns
- Direct assignment
- Conditional updates with loc
- Vectorized transformations
- Creating new columns

### 06 — Adding & Removing Rows/Columns
- Adding columns
- Dropping columns and rows
- Structural DataFrame modifications
- concat for row addition

### 07 — Sorting Data
- sort_values()
- sort_index()
- Multi-column sorting
- Handling NaN during sorting

### 08 — GroupBy & Aggregation
- Basic groupby operations
- Multiple aggregations
- Named aggregations
- transform vs aggregate
- Group filtering

### 09 — Data Cleaning & Type Casting
- Inspecting dtypes
- astype()

- fillna() vs dropna()
- Datetime conversion

### 10 — DateTime & Time Series
- pd.to_datetime()
- Extracting date components
- Date filtering
- Resampling
- Rolling window calculations

### 11 — Reading/Writing Data (I/O)
- CSV
- Excel
- JSON
- SQL (SQLite)
- Exporting data to different formats
